====== Rspec on the command-line ======
**Command Line:**
$//rails generate rspec install											//<<- tells rails to go to spec/ for generate commands and makes .rspec file in root 




**Setup:**
--																		<<- in config/application.rb to make generators generate rspec, not test-unit
config.generators do |g|
  g.test_framework :rspec
end
--
