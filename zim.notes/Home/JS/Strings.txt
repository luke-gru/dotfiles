====== Strings ======

**Methods:**
toLowerCase()


//Regex matching methods://
string.search(/regex/flag);							 => doesn't support g-flag, converts non-regexes to regexes

string.replace(/regex/flag, "string")				 => default non-global
string.replace("string", "string")					 => doesn't convert first string to regex
string.replace(/regex/, "$1")						=> first match saved
string.replace(/regex/, function() )					=> 

string.match(/regex/flag)							=> returns ["array", "of", "matching", "elements"]  (default non-global)
											  => when non-global array[0] will be matched substring, a[1] will be first subexpression match, a[2]... 

												Also, array.input will be input string, array.index will be index of match of array[0]



string.split(/,/)										=> array of elements (split can take a string like in Ruby)


**Quirks:**
Strings are immutable! They are compared by //value.//
