====== Arguments property object ======

**What is it?**
An object that's the property of the call object that changes depending on the function and arguments given.
An 'array' that has the number of elements corresponding to the number of arguments that the invocation of the function took.

arguments.length will have the value of 2 is the function is invoked with 2 arguments.

//Example://
function max( )
{
var m = Number.NEGATIVE_INFINITY;
// Loop through all the arguments, looking for, and
// remembering, the biggest
for(var i = 0; i < arguments.length; i++)
if (arguments[i] > m) m = arguments[i];
// Return the biggest
return m;
}

**Quirk:**
Changing the value of an argument with an argument name changes the value that is retrieved through the arguments[] array.

function f(x) {
alert(x);
arguments[0] = null;
alert(x);				=> displays "null"
}

**Callee property:**
Allows an unnamed function to invoke itself recursively.

function(x) {
if (x <= 1) return 1;
return x * arguments.callee(x-1);
}
