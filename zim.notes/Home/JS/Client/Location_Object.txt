====== Location Object ======

**Properties:**
protocol
href							=> full URL w/ protocol					=> location string as //requested! //document.URL is location string //as received (ex: redirect)//
pathname						=> part after first "/" up until first "?"
search							=> part after and including "?"

location = "http://www.google.com" is the same as location.href = ...

location="/help" goes to that path on the current domain

**Methods:**
reload()
replace()					=> loads and replaces URL, and replaces this URL for the one in the history list
