====== !operator in front of variables ======

>$ user = 3
>>3
>$ user.nil?
>>false
>$ !user.nil?
>>true

**Useful for seeing if users are signed in:**

def signed_in?
	!current_user.nil?

end

username.signed_in? >> true if they are a current user
