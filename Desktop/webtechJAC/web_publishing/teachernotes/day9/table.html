Notes:

  • All attributes of the table tag are optional.
  • The caption,thead, tfoot, and tbody tags are optional, but if you do use them it must be in that order.
  • The th tag can be used in the thead or tbody areas.
  • Rowspan and colspan must be specified in the first cell of the group to be merged.
  • The align and valign attributes can be used in either the tr (for the whole row) or the td/th tags (for a specific cell)
